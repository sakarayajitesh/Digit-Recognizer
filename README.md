# Digit-Recognizer
Building a MNIST Digit Recognizer using **Deep Learning**.
\
\
\
!["MNIST Dataset"](/mnist_image.png "MNIST Dataset")



# Overview
The model predicts the hand written digits from the MNIST dataset.

* **Model : 784 -> 400 -> 1**
* **Optimizer : SGD**
* **Loss : Cross Entropy**
* **Accuracy : 96.5%**
